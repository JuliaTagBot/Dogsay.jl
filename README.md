# Dogsay.jl

[![Build Status](https://travis-ci.com/r0cketr1kky/Dogsay.jl.svg?branch=master)](https://travis-ci.com/r0cketr1kky/Dogsay.jl)

## Installation

```julia
Pkg.add("Dogsay")
```

## Usage

In Julia

```julia
using Dogsay
dogsay("Woof!")
```
